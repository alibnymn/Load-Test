Repo ini berisi script converter Node.js untuk mengubah file laporan JSON standar dari **Artillery.io** menjadi Dashboard HTML yang interaktif, modern, dan mudah dibaca. 

Dashboard ini dirancang khusus untuk kebutuhan QA Reporting dengan tampilan *Artillery Cloud Style*.

## 📊 Fitur Utama
- **Load Summary Bar**: Ringkasan VUsers, Success Rate, Avg RPS, dan Peak RPS di bagian paling atas.
- **Detailed Metrics Cards**: Detail Latency (Min, Mean, Max, p95, p99) dalam bentuk card modern.
- **Trend Charts**: Grafik fluktuasi Latency dan Throughput (RPS) dari waktu ke waktu.
- **Glossary Section**: Penjelasan istilah metrik untuk memudahkan stakeholder non-teknis.
- **Linear Layout**: Desain satu kolom yang responsif dan nyaman di-*scroll*.

## 🛠️ Prasyarat
Pastikan Anda sudah menginstal:
- [Node.js](https://nodejs.org/) (Versi 14 ke atas)
- [Artillery](https://www.artillery.io/) (`npm install -g artillery`)

## 🚀 Cara Penggunaan

### 1. Jalankan Load Test
Jalankan pengujian Artillery Anda dan simpan hasilnya ke dalam file `report.json`:
```bash
artillery run -o report.json script-test.yaml
```

### 2. Konversi ke HTML
Pastikan file `converter.js` dan `report.json` berada di folder yang sama, lalu jalankan:
```bash
node converter.js
```

### 3. Lihat Hasil
Buka file `my-report.html` yang muncul di folder Anda menggunakan browser (Chrome/Edge/Firefox).

## 📁 Struktur File
- `script-test.yaml`: Skenario pengujian Artillery Anda.
- `report.json`: Data mentah hasil test (Generated by Artillery).
- `converter.js`: Script Node.js untuk memproses data menjadi HTML.
- `my-report.html`: Hasil akhir laporan dashboard.

## 📝 Penjelasan Metrik Utama
| Metrik | Deskripsi |
| :--- | :--- |
| **VUsers Created** | Total simulasi pengguna unik selama test. |
| **Avg Req/s** | Rata-rata jumlah permintaan per detik (Throughput). |
| **p95 Latency** | Ambang batas waktu respon untuk 95% total request (Standard QA). |
| **Success Rate** | Persentase skenario yang berhasil diselesaikan tanpa error. |

## 🎨 Kustomisasi Tampilan
Anda dapat mengubah warna aksen atau font di dalam variabel `:root` pada file `converter.js`:
- `--blue`: Warna utama untuk RPS & Link.
- `--yellow`: Warna peringatan untuk p95.
- `--red`: Warna kritikal untuk p99 & Failure.
